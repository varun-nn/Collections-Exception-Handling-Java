              # Collections & Exception Handling

## Objective
Demonstrate Java Collections and Exception Handling.

## Collections Used

- ArrayList (Store student names)
- HashMap (Store student marks)
- HashSet (Store unique courses)

## Exception Handling

- try-catch block
- Custom Exception (InvalidAgeException)
- InputMismatchException
- finally block
